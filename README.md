# 💕 浪漫婚纱照3D展示应用

一个精美的婚纱照展示应用，具有3D效果和优雅的动画。

## ✨ 特性

- 🎨 **精美的UI设计** - 渐变背景和流畅动画
- 🎭 **3D展示效果** - 使用Three.js实现的3D照片展示
- 📱 **响应式设计** - 完美适配手机、平板和电脑
- 🖼️ **照片画廊** - 网格布局展示所有照片
- ⌨️ **键盘导航** - 支持方向键和ESC键
- 👆 **触摸支持** - 移动设备滑动切换照片
- ✨ **粒子效果** - 浪漫的浮动粒子背景

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 准备照片

将您的婚纱照放在 `photos` 文件夹中（如果不存在会自动创建）。

支持的图片格式：`.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.bmp`

### 3. 启动服务器

```bash
npm start
```

### 4. 打开浏览器

访问 `http://localhost:3000` 即可看到您的婚纱照展示！

## 📁 项目结构

```
wedding/
├── server.js          # Express后端服务器
├── package.json       # 项目配置和依赖
├── photos/            # 照片文件夹（需要您创建并放入照片）
└── public/            # 前端文件
    ├── index.html     # 主页面
    ├── style.css      # 样式文件
    └── app.js         # JavaScript逻辑
```

## 🎮 使用说明

1. **浏览照片**：在主页面的网格中浏览所有照片
2. **查看大图**：点击任意照片进入3D查看模式
3. **切换照片**：
   - 点击左右箭头按钮
   - 使用键盘方向键（← →）
   - 在移动设备上左右滑动
4. **关闭查看器**：点击右上角的 × 或按 ESC 键

## 🎨 自定义

### 修改照片文件夹路径

编辑 `server.js` 文件中的 `PHOTOS_DIR` 变量：

```javascript
const PHOTOS_DIR = path.join(__dirname, 'your-photos-folder');
```

### 修改端口

编辑 `server.js` 文件中的 `PORT` 变量：

```javascript
const PORT = 3000; // 改为您想要的端口
```

## 💡 提示

- 建议照片文件名使用有意义的名称，便于识别
- 照片会自动按文件名排序
- 首次运行如果没有照片文件夹，程序会自动创建
