# 婚纱照展示 - 便携版使用说明

## 一、打包步骤（在本机执行一次）

1. 安装依赖：
   ```bash
   npm install
   ```
2. **要在「没有 Node 的 Windows 电脑」上运行（推荐，在 Mac 上即可打包）：**
   ```bash
   npm run pack:win-portable
   ```
   会生成 **`wedding-portable-win/`** 文件夹，内含 Windows 版 Node + 项目文件。把该文件夹拷到 U 盘，在 Windows 上解压后双击 **`run.bat`** 即可运行，无需在 Windows 上安装 Node。

3. **仅 Mac 便携包（单文件可执行）：**
   ```bash
   npm run pack
   npm run pack:full
   ```
   得到 `wedding-portable/`，内含 `wedding-photo-gallery-macos`。
   **在 Windows 上要 exe 单文件**：需在 Windows 电脑上执行 `npm run pack:win`，得到 `dist/wedding-photo-gallery-win.exe`。

## 二、带到其他电脑的文件夹结构

把下面这些**放在同一文件夹**里，整个文件夹拷贝到 U 盘或其它电脑即可：

```
你的文件夹/
├── wedding-photo-gallery-win.exe   （Windows 用）
  或 wedding-photo-gallery-macos    （Mac 用）
├── photos/          ← 把婚纱照图片放这里（.jpg / .png / .gif / .webp）
└── music/           ← 把背景音乐 .mp3 放这里
```
（可执行文件与 photos、music 必须在同一目录。）

- **不需要**安装 Node.js，直接运行可执行文件即可。
- `photos`、`music` 若不存在，程序会自动创建空文件夹；你只要把照片和音乐放进去即可。

## 三、在其他电脑上运行

- **Windows**：双击 `wedding-photo-gallery-win.exe`。
- **macOS**：在终端执行（或右键“打开”）：
  ```bash
  ./wedding-photo-gallery-macos
  ```
  若提示“无法打开，因为无法验证开发者”，可到 **系统设置 → 隐私与安全性** 中允许该应用。

运行后：

- 打开浏览器访问：**http://localhost:3000**
- 若本机有防火墙提示，选择“允许访问”。

## 四、照片与音乐说明

- **照片**：支持 `.jpg`、`.jpeg`、`.png`、`.gif`、`.webp`、`.bmp`，横版会单页或双页展示，竖版会参与组合。
- **音乐**：仅支持 `.mp3`，可放多首，程序会随机循环播放；首次打开页面需在页面内点击一下才会开始播放。

---

如有问题，可先确认：可执行文件与 `photos`、`music` 是否在同一目录，以及 3000 端口是否被占用。
