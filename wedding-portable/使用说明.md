# 婚纱照展示 - 便携版使用说明

## 一、打包步骤（在本机执行一次）

1. 安装依赖并打包成可执行文件：
   ```bash
   npm install
   npm run pack
   ```
2. 打包完成后，在项目根目录会生成 `dist` 文件夹。
   - **在 Mac 上打包**：得到 `wedding-photo-gallery-macos`（无扩展名的可执行文件）。
   - **在 Windows 上打包**：需把 `package.json` 里 `pack` 脚本中的 `node18-macos-x64` 改成 `node18-win-x64`，并把 `--output dist/wedding-photo-gallery` 改成 `--output dist/wedding-photo-gallery-win.exe`，再执行 `npm run pack`，得到 `wedding-photo-gallery-win.exe`。

## 二、带到其他电脑的文件夹结构

把下面这些**放在同一文件夹**里，整个文件夹拷贝到 U 盘或其它电脑即可：

```
你的文件夹/
├── wedding-photo-gallery-win.exe   （Windows 用）
  或 wedding-photo-gallery-macos    （Mac 用）
├── photos/          ← 把婚纱照图片放这里（.jpg / .png / .gif / .webp）
└── music/           ← 把背景音乐 .mp3 放这里
```
（可执行文件与 photos、music 必须在同一目录。）

- **不需要**安装 Node.js，直接运行可执行文件即可。
- `photos`、`music` 若不存在，程序会自动创建空文件夹；你只要把照片和音乐放进去即可。

## 三、在其他电脑上运行

- **Windows**：双击 `wedding-photo-gallery-win.exe`。
- **macOS**：在终端执行（或右键“打开”）：
  ```bash
  ./wedding-photo-gallery-macos
  ```
  若提示“无法打开，因为无法验证开发者”，可到 **系统设置 → 隐私与安全性** 中允许该应用。

运行后：

- 打开浏览器访问：**http://localhost:3000**
- 若本机有防火墙提示，选择“允许访问”。

## 四、照片与音乐说明

- **照片**：支持 `.jpg`、`.jpeg`、`.png`、`.gif`、`.webp`、`.bmp`，横版会单页或双页展示，竖版会参与组合。
- **音乐**：仅支持 `.mp3`，可放多首，程序会随机循环播放；首次打开页面需在页面内点击一下才会开始播放。

---

如有问题，可先确认：可执行文件与 `photos`、`music` 是否在同一目录，以及 3000 端口是否被占用。
